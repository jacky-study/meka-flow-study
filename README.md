# meka-flow-study

> 深入研究 [meka-flow](https://github.com/osvald-go2/meka-flow) 开源项目的技术实现

## 项目简介

Meka Flow 是一个工作流引擎，本项目通过源码阅读和实验，深入理解其架构设计与核心实现。

## 目录结构

```
meka-flow-study/
├── meka-flow/       # Meka Flow 源码
├── notes/           # 学习笔记
├── scripts/         # 实验脚本
└── RESEARCH-LOG.md  # 研究日志
```

## 学习内容

- 工作流引擎设计原理
- 流程编排与调度机制
- 可扩展架构模式
